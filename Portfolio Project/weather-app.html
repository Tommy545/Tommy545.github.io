Weather Application


I felt like making a Weather Application


How the App works

The Weather app makes use of a free API that openweathermap provides, this is my second time utilising a API which I have to say that this one was easier, but to be fair it was a smaller project.


When you first open the website/app you'll arrive with this prompt, you can fill out any known location (it can be larger or smaller then a city) and by pressing search or your enter key you will see the weather in that area



Made with HTML, CSS & JavaScript
