Discord Bot Capture the Flag Project

For this project I used to play a game in my freetime that wasn't really that good. So I decided to make my own little mini game within it.


How it works

I've made a few commands to make this project work:

- .Map

- .Attack

- .Done


.Map:

This command shows you the map which gets updated everytime a point is captured by a different team or if I would want to reset them to all white.


.Attack

On this page you'd be able to see the families that have signed up for the foodbank and if you're a high enough rank you'd be able to add some, its accompanied with a search bar on the left that works with javascript AJAX to talk with the database via php.


.Done

This command would be used after using .Attack


Closing Statement:

I really liked this project as I learned how to do stuff like link php and JavaScript. it was very educational.
