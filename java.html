Reaction Speed Game


This game was a school project and I found this idea quite cool, I've worked on this for a while, it was back when I was still being introduced to coding outside of Scratch and Python.


How the game works

when you first enter you will see this picture:


Pressing on the cube will start off with two things: a timer and a sequence of cubes. Every time you press on a cube, a new one will spawn on the canvas at a random location. This will repeat till the timer reaches 10 seconds. Once it reaches its end, the game round is over, and a count of the number of cubes you've hit in that time frame will show.
